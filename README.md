# 💳 Credit Wise Loan Prediction

## 📌 Overview

This project predicts whether a loan application should be approved or rejected based on applicant details using Machine Learning techniques.
Multiple models were trained and compared to select the best performing model.

---

## 🚀 Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Model Training & Comparison
* Loan Approval Prediction System

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

---

## 📊 Models Used

* Logistic Regression
* K-Nearest Neighbors (KNN)
* Naive Bayes

---

## 🏆 Best Model

Naive Bayes performed best among all models:

* Accuracy: **86.5%**
* Precision: **~0.80**
* Recall: **~0.73**
* F1 Score: **~0.76**
